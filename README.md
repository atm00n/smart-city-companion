# Run and deploy your City Wander app

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Download and unzip the folder   
2. open unzipped folder in Visual Studio Code
3. Open terminal and type 'npm install' to install npm dependencies
4. Type 'npm run dev'
5. Hover over local host then click on follow link
6. Your website is running locally
